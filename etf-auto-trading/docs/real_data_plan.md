## 🎯 최종 선택: 한국투자증권 OpenAPI (Only)

### ✅ 한투 API 선택 이유  
- **완벽 호환**: Linux + Python + Oracle
- **무료**: 계좌만 있으면 OK
- **안정성**: REST API 표준 방식
- **확장성**: 향후 기능 추가 용이

### 📋 준비 과정
1. **계좌 개설** (온라인 가능)
2. **OpenAPI 신청** → https://apiportal.koreainvestment.com
3. **APP_KEY, SECRET 발급** (승인 후)
4. **샌드박스 테스트** (모의투자)
5. **실계좌 연동**

### 🔧 기술적 구현
```python
# 실시간 ETF 데이터 수집
def collect_real_etf_data():
    etf_codes = ['069500', '229200', '102110', ...]  # 10개 ETF
    
    for code in etf_codes:
        # 1. 현재가 조회
        price_data = kis_api.get_current_price(code)
        
        # 2. 일봉 데이터 조회  
        ohlcv_data = kis_api.get_daily_ohlcv(code)
        
        # 3. Oracle DB 저장
        save_to_database(code, price_data)
        
        # 4. 기술적 지표 계산
        calculate_technical_indicators(code)
        
        # 5. 매매 신호 생성
        generate_trading_signals(code)
```

### 📊 일일 조회 한도 관리
- **현재가 조회**: 10개 ETF × 1일 = 10건
- **일봉 데이터**: 10개 ETF × 1일 = 10건  
- **과거 데이터**: 최초 1회만 (180일 × 10개 = 1,800건)
- **총 사용량**: 하루 20건 (한도 1만건의 0.2%)
