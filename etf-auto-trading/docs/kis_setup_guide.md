# 한국투자증권 OpenAPI 연동 완전 가이드
*키움증권 등 다른 API 제외, 한투 API에만 집중*

## 🎯 왜 한국투자증권 OpenAPI인가?

### ✅ 우리 시스템과 완벽 호환
- **REST API**: 표준 웹 API 방식
- **Linux 지원**: 우리 서버(Rocky Linux) 완벽 호환
- **Python 친화적**: 기존 코드와 자연스러운 연동
- **Oracle 연동**: DB 저장까지 seamless

### ❌ 키움증권 API 제외 이유
- **OCX 기반**: Windows 전용 (우리는 Linux)
- **복잡한 설정**: 개발/운영 환경 분리 어려움
- **호환성 문제**: 현재 인프라와 맞지 않음
