# 🎨 디스코드 알림 시스템 (단일 알림 플랫폼)

## 📋 개요

**ETF 자동매매 시스템은 디스코드 웹훅을 통해 모든 알림을 전송합니다.**

### ✅ **디스코드 선택 이유**
- **🎨 예쁜 임베드 메시지**: 색상과 이모지로 직관적인 정보 전달
- **⚡ 실시간 알림**: 웹훅을 통한 즉시 전송 (지연 없음)
- **📱 모바일 최적화**: 디스코드 앱으로 언제든 확인 가능
- **🔧 설정 간편**: 봇 없이 웹훅만으로 완료 (5분 설정)
- **👥 팀 공유**: 서버 공유로 가족/친구와 정보 공유
- **🔒 보안**: 웹훅 URL만 안전하게 관리하면 완료

### ❌ **제거된 알림 방식**
- **텔레그램**: 복잡한 봇 설정, 제한적인 메시지 포맷
- **이메일**: 실시간성 부족, 스팸 필터링 위험
- **SMS**: 비용 발생, 메시지 길이 제한

## 🚀 5분 설정 가이드

### **1️⃣ 디스코드 웹훅 생성**

#### **Step 1: 디스코드 서버 준비**
1. 디스코드에서 새 서버 생성 (또는 기존 서버 사용)
2. 서버 이름 예시: `ETF 자동매매` 또는 `투자봇`

#### **Step 2: ETF 알림 채널 생성**  
1. 서버에서 텍스트 채널 추가
2. 채널 이름 예시: `#etf-signals` 또는 `#매매신호`

#### **Step 3: 웹훅 설정**
1. 채널 우클릭 → **"채널 편집"** 클릭
2. 좌측 메뉴 → **"연동"** → **"웹후크"** 선택
3. **"새 웹후크"** 버튼 클릭
4. 웹훅 설정:
   - **이름**: `루나 ETF Bot 🌙`
   - **아바타**: 원하는 이미지 (선택사항)
5. **"웹후크 URL 복사"** 클릭 → URL 저장

```
예시 웹훅 URL:
https://discord.com/api/webhooks/1234567890123456789/abcdefghijklmnopqrstuvwxyz-1234567890
```

### **2️⃣ ETF 시스템 연동**

#### **환경변수 설정**
```bash
# .env 파일 생성/수정
cd /root/.openclaw/workspace/etf_collector
cp .env.example .env
nano .env

# 디스코드 웹훅 URL 입력
DISCORD_WEBHOOK_URL=https://discord.com/api/webhooks/YOUR_WEBHOOK_URL
```

#### **설정 확인**
```bash
# config.yaml 확인 (이미 설정됨)
grep -A 10 "discord:" config.yaml

# 출력 예시:
# discord:
#   enabled: true
#   webhook_url: "${DISCORD_WEBHOOK_URL}"
#   signal_threshold: "medium"
```

### **3️⃣ 연결 테스트**

```bash
# 디스코드 알림 테스트
python3 etf_discord_notifier.py

# 웹훅 URL 입력 프롬프트에 복사한 URL 붙여넣기
# ✅ 테스트 메시지가 디스코드 채널에 나타나면 성공!
```

## ⚙️ 고급 설정

### **🎯 알림 필터링**

#### **신호 강도별 알림 설정**
```yaml
# config.yaml
notifications:
  discord:
    signal_threshold: "strong"  # 강한 신호만 알림
```

- `weak`: 모든 신호 알림
- `medium`: 중간 이상 신호만 알림 (기본값)  
- `strong`: 강한 신호만 알림 (추천)

#### **알림 타입별 활성화/비활성화**
```yaml
# config.yaml
notifications:
  discord:
    send_market_status: true     # 시장 개장/마감 알림
    send_daily_summary: true     # 일일 요약 알림
    send_system_alerts: true     # 시스템 오류/경고 알림
```

### **🎨 메시지 커스터마이징**

#### **봇 이름 및 아바타 변경**
```python
# etf_discord_notifier.py 수정
payload = {
    'username': '나만의 ETF Bot 🤖',
    'avatar_url': 'https://your-avatar-url.com/image.png'

```

#### **알림 색상 변경**
```python
# 색상 코드 변경 (16진수)
'BUY': {
```

## 📱 모바일 최적화

### **🔔 푸시 알림 설정**
1. 디스코드 모바일 앱 설치
2. 설정 → 알림 → 푸시 알림 활성화
3. 서버별 알림 설정에서 ETF 서버 알림 ON

### **📲 중요 알림만 받기**
1. ETF 채널에서 🔔 아이콘 클릭
2. `@mentions only` 선택
3. 중요한 신호만 멘션으로 전송되도록 설정

### **🌙 야간 알림 제어**
```python
# 야간 시간대 알림 차단 (옵션)
def is_night_time():
    now = datetime.now()
    return 23 <= now.hour or now.hour <= 7

if not is_night_time():
    send_discord_notification()
```

## 🛠️ 문제 해결

### **❌ 알림이 오지 않을 때**

#### **1. 웹훅 URL 확인**
```bash
echo $DISCORD_WEBHOOK_URL
# URL이 올바른지 확인 (https://discord.com/api/webhooks/...)
```

#### **2. 연결 테스트**
```bash
python3 etf_discord_notifier.py
# 테스트 메시지가 디스코드에 나타나는지 확인
```

#### **3. 설정 확인**
```bash
grep -A 10 "discord:" config.yaml
# enabled: true 인지 확인
```

#### **4. 로그 확인**
```bash
tail -20 logs/etf_redis.log | grep -i discord
# 디스코드 관련 오류 메시지 확인
```

### **🔧 일반적인 해결 방법**

#### **웹훅 URL 오류**
- 웹훅이 삭제되었거나 만료됨
- 새 웹훅 생성 후 URL 업데이트

#### **권한 문제**
- 디스코드 서버에서 웹훅 권한 확인
- 채널 접근 권한 확인

#### **네트워크 문제**
- 방화벽에서 Discord 도메인 허용 확인
- 프록시 설정 확인 (필요시)
