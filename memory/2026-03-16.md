## postgres01 (50.16) SSH 문제 해결
- 원인: systemd-logind가 SSH 세션 생성 시 "opening" 상태에서 멈추는 버그 (OEL9)
- 해결: systemd-logind mask + pam_systemd 주석 처리 (password-auth, system-auth, sshd)
- 추가: GSSAPIAuthentication no, UseDNS no, MaxStartups 50:30:100
- firewalld disable
