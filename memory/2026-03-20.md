### 교훈
- Claude Code가 root에서 `--permission-mode bypassPermissions` 실행 불가 (--dangerously-skip-permissions 거부)
- 부팅 시퀀스 크래시 원인: setInterval 클로저에서 `allLines[idx]` → idx++ 후 참조하면 undefined → `.includes()` TypeError
  - 해결: `const line = allLines[idx]; idx++; setLines(prev => [...prev, line])` 순서로 변경
- Tailwind 커스텀 클래스(text-neon-green 등)보다 인라인 style이 SSR/하이드레이션 안전
- 부팅 시퀀스는 별도 컴포넌트(boot-screen.tsx)로 분리해야 안정적
- sessionStorage 접근은 useEffect 안에서만 (SSR 방어)
- ps aux는 커맨드라인 잘림 → ps -ef 사용해야 전체 vmx 경로 보임
