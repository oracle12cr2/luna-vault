- **OPEN 계정만 수집**: `dba_users` 테이블과 JOIN하여 `account_status = 'OPEN'` 조건 추가
- **SYS, SYSTEM 제외**: 기존 조건 유지
- **적용 대상**: High CPU SQL, High Buffer Gets SQL, SQL Tuning Advisor 권장사항

**수정된 쿼리 구조**:
```sql
FROM v$sql s, dba_users u
WHERE s.parsing_schema_name = u.username
  AND u.account_status = 'OPEN'  -- 새로 추가
  AND u.username NOT IN ('SYS', 'SYSTEM', 'DBSNMP')
```
