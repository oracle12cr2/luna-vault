# 🎨 디스코드 알림 설정 가이드

## 📋 개요

ETF 자동매매 시스템의 매매 신호를 **디스코드 웹훅**으로 실시간 알림받는 방법을 설명합니다.

### 🎯 디스코드 알림 장점
- **🎨 예쁜 임베드 메시지**: 색상과 이모지로 직관적인 신호 전달
- **⚡ 즉시 알림**: 웹훅을 통한 실시간 전송
- **📱 모바일 지원**: 디스코드 앱으로 언제든 확인
- **🔧 간단한 설정**: 봇 없이도 웹훅만으로 알림 가능

---

## 🚀 1단계: 디스코드 웹훅 생성

### **📱 디스코드 서버 준비**

1. **디스코드 서버 생성** (없는 경우)
   - 디스코드 앱 또는 웹사이트 접속
   - `+` 버튼 → "서버 만들기"
   - 서버 이름: `ETF 투자봇` 또는 원하는 이름

2. **ETF 알림 채널 생성**
   - 서버 내에서 채널 추가 (`+` 버튼)
   - 채널 이름: `#etf-signals` 또는 `#투자알림`
   - 채널 타입: 텍스트 채널

### **🔧 웹훅 생성**

1. **채널 설정 들어가기**
   - ETF 알림 채널에서 우클릭
   - "채널 편집" 선택

2. **웹훅 만들기**
   - 좌측 메뉴에서 "연동" 클릭
   - "웹후크" 탭 선택
   - "새 웹후크" 버튼 클릭

3. **웹훅 설정**
   - 이름: `루나 ETF Bot` 🌙
   - 아바타: 원하는 이미지 업로드 (선택사항)
   - "웹후크 URL 복사" 클릭 → **중요: 이 URL을 저장!**

```
예시 웹훅 URL:
https://discord.com/api/webhooks/1234567890/abcdefghijklmnopqrstuvwxyz
```

## ⚙️ 2단계: ETF 시스템 설정

### **📄 환경변수 설정**

#### 방법 1: .env 파일 (추천)
```bash
# ETF 시스템 디렉토리에서
cp .env.example .env
nano .env

# 디스코드 웹훅 URL 입력
DISCORD_WEBHOOK_URL=https://discord.com/api/webhooks/YOUR_WEBHOOK_URL
```

#### 방법 2: 시스템 환경변수
```bash
export DISCORD_WEBHOOK_URL="https://discord.com/api/webhooks/YOUR_WEBHOOK_URL"
```

### **📝 config.yaml 수정**
```yaml
notifications:
  discord:
    enabled: true  # false → true로 변경
    webhook_url: "${DISCORD_WEBHOOK_URL}"
    signal_threshold: "medium"  # weak, medium, strong
    send_market_status: true    # 시장 개장/마감 알림
    send_daily_summary: true    # 일일 요약
```

### **🔍 웹훅 테스트**

## 🔧 5단계: 고급 설정

### **📊 알림 필터링**

#### **신호 강도별 필터링**
```yaml
# config.yaml
notifications:
  discord:
    signal_threshold: "strong"  # strong 신호만 알림
```

- `weak`: 모든 신호 알림
- `medium`: 중간 이상 신호만 알림  
- `strong`: 강한 신호만 알림 (추천)

### **🎨 메시지 커스터마이징**

#### **봇 이름 변경**
```python
# etf_discord_notifier.py 수정
payload = {
    'username': '나만의 ETF Bot 🤖',  # 원하는 이름
    'avatar_url': 'https://your-avatar-url.com/image.png'
}
```

#### **색상 변경**
```python
# 매수 신호 색상 (초록색 → 파란색)
'BUY': {
    'color': 0x0099ff,  # 파란색
    'emoji': '🔵',
    'title': '매수 기회'
}
```

### **⚠️ 문제 해결**

#### **알림이 오지 않을 때**
```bash
# 1. 웹훅 URL 확인
echo $DISCORD_WEBHOOK_URL

# 2. 연결 테스트
python3 etf_discord_notifier.py

# 3. 설정 확인
grep -A 10 "discord:" config.yaml

# 4. 로그 확인
tail -20 logs/etf_redis.log | grep -i discord
```

#### **메시지가 깨질 때**
- 웹훅 URL이 올바른지 확인
- 디스코드 서버 권한 확인
- 채널이 삭제되지 않았는지 확인
