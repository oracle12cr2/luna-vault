# Oracle ADG (Active Data Guard) - READ ONLY WITH APPLY 설정

## 목적
Physical Standby DB를 MOUNTED 상태에서 READ ONLY WITH APPLY 모드로 전환하여
실시간 리두 로그 적용(MRP0)을 유지하면서 읽기 쿼리도 가능하게 한다.

## 사전 조건 확인
```sql
-- 현재 상태 확인
SELECT DATABASE_ROLE, OPEN_MODE FROM V$DATABASE;
SELECT PROCESS, STATUS, SEQUENCE# FROM V$MANAGED_STANDBY WHERE PROCESS LIKE 'MRP%';
```

정상 시작 조건:
- PHYSICAL STANDBY / MOUNTED ✅
- MRP0 APPLYING_LOG (Sequence# 361 이상) ✅

## 절차

```sql
-- 1. MRP 중지
ALTER DATABASE RECOVER MANAGED STANDBY DATABASE CANCEL;

-- 2. READ ONLY 오픈
ALTER DATABASE OPEN READ ONLY;

-- 3. APPLY 재시작
ALTER DATABASE RECOVER MANAGED STANDBY DATABASE USING CURRENT LOGFILE DISCONNECT;

-- 4. 상태 확인
SELECT DATABASE_ROLE, OPEN_MODE FROM V$DATABASE;
SELECT PROCESS, STATUS, SEQUENCE# FROM V$MANAGED_STANDBY WHERE PROCESS LIKE 'MRP%';
```

## 성공 기준
- `OPEN_MODE = READ ONLY WITH APPLY`
- MRP0 다시 `APPLYING_LOG` 상태로 복귀

## 환경
- 서버: oracle19cadg (192.168.50.41)
- DB: Oracle 19c Physical Standby
- Primary: oracle19c-scan (192.168.50.27~29)
