# 임금체불 민사소송 준비

## 사건 개요
- **피의자(고용주):** 이옥희
- **형사 사건번호:** 의정부지방검찰청 고양지청 2025형제17206호
- **형사 처분:** 구약식 (벌금형) — 2025.07.10 처분
- **체불 금액:** 400만원 이상 (상세 확인 필요)

## 소멸시효
- **임금채권 소멸시효: 3년** (근로기준법 제49조)
- 기산점: 각 월급의 지급일 다음 날부터 3년
- ⚠️ 시효 만료 전에 소장 접수해야 함

## 민사소송 진행 방법
1. **관할법원:** 의정부지방법원 고양지원 (피고 소재지 기준)
2. **소장 작성:** 임금청구소송 (나홀로소송 가능)
3. **인지대:** 소송가액 400만원 기준 약 3~4만원
4. **송달료:** 약 5만원
5. **형사 구약식 = 유죄 인정 → 민사에서 매우 유리 (사실상 승소)**

## 필요 서류 (준비할 것)
- [ ] 근로계약서 (없어도 가능)
- [ ] 급여명세서 / 통장거래내역 (입금 기록)
- [ ] 형사 처분 결과 (알림 캡처 보관)
- [ ] 체불 금액 상세 내역 (몇 월~몇 월, 각 금액)
- [ ] 사업장 정보 (상호, 주소, 사업자등록번호)
- [ ] 고용주 정보 (이옥희, 주소)

## 확인 필요 사항
- [ ] 체불 기간 (몇 년 몇 월 ~ 몇 년 몇 월)
- [ ] 정확한 체불 금액 (월별 내역)
- [ ] 고용주 현재 주소
- [ ] 노동청 진정 이력 여부
- [ ] 와이프와 상세 내용 확인 후 정리

## 참고
- 형사(벌금)와 민사(임금 청구)는 별개 — 벌금 내도 밀린 월급은 따로 받아야 함
- 소장 작성 도움 필요하면 유나한테 요청
