# Google Stitch 대규모 업데이트 요약

> 출처: https://www.youtube.com/watch?v=-vsUVH9nVwc
> 정리일: 2026-03-21

## 핵심 한 줄

Google Stitch에서 디자인 → `design.md` 내보내기 → Claude Code에서 바로 서비스 구현하는 워크플로우가 가능해졌다.

## 주요 변경점

- **UI 전면 개편** — 무한 캔버스 방식, 하단 입력창, 오른쪽 도구 패널
- **모델 선택, 음성 명령(라이브 모드), 파일/URL 업로드** 지원
- 앱/웹 두 가지 모드로 프로젝트 시작 가능

## 핵심 신기능: `design.md` 내보내기

- 내보내기 메뉴에 **"프로젝트 요약"** 항목 추가
- HTML, PNG 스크린샷, **`design.md`** 파일이 포함된 ZIP 생성
- `design.md`는 색상·컴포넌트·레이아웃 등 디자인 명세가 정리된 마크다운
- **Claude Code에 바로 투입 가능**

## 시연 흐름

1. 패션 코디 추천 AI 서비스의 **PRD 파일을 Stitch에 업로드**
2. Stitch가 컬러 무드보드 자동 생성 + **5개 페이지 디자인** 완성
3. 인스턴트 프로토타입 변환 (모바일/태블릿/웹 반응형 미리보기)
4. `design.md`로 내보내기
5. `prd.md` + `design.md`를 Claude Code에 넣고 빌드
6. Gemini API 연동 → 사진 업로드 시 스타일 분석 + 제품 추천까지 완성

## 포인트

- Stitch는 **무료**
- 디자인 경험 없어도 **바이브 디자인 → 바이브 코딩** 전체 흐름을 혼자 완성 가능
- PRD 작성 → Stitch 디자인 → Claude Code 구현 흐름으로 프로토타입 속도 극대화
