람인 개발자가 SQL 튜닝을 하는 것이 맞습니다. 저 또한 이러한 상햄l서 SQL 튜닝 
학습을 시작하였습니다 시중에 있는 SQL 튜닝 관련 책을 보고 학습하면서 다음과 같 
은힘든점이 있었습니다
