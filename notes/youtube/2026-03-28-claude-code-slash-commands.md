# 클로드 코드 슬래시 커맨드 완전 정복

> 출처: https://www.youtube.com/watch?v=JUiA0bG7hOQ

## 핵심 메시지
자주 쓰는 프롬프트를 매번 타이핑하지 말고, 슬래시 커맨드로 저장해서 한 번에 실행하자!

## 내장 커맨드 정리

| 커맨드 | 용도 |
|--------|------|
| `/clear` | 대화 컨텍스트 초기화 |
| `/resume` | 이전 세션 이어가기 |
| `/model` | 모델 선택 (비용↔성능 조절) |
| `/diff` | 파일 변경사항 확인 |
| `/status` | 버전 + MCP 목록 확인 |
| `/usage` | 이번 주 사용량/토큰 소모량 |
| `/btw` | 작업 중 즉석 질문 (가장 자주 사용!) |
| `/review` | 코드 리뷰 + 보안 점검 |
| `/context` | 컨텍스트 토큰 시각화 |

## 커스텀 커맨드 만들기

- **마크다운 파일 1개 = 커맨드 1개** (파일명 = 커맨드명)
- 저장 위치: `~/.claude/commands/`
- 재시작하면 `/morning-brief` 처럼 바로 사용 가능

### 예시: morning-brief 커맨드
오늘 작업 현황 브리핑, 최신 커밋 3개 확인을 자동 실행

### 활용 아이디어
- `/summarize` — 요약 자동화
- `/linkedin` — LinkedIn 포스트 작성
- 자신만의 반복 업무를 커맨드화
