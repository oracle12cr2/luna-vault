# 이력서 - 기술 스킬 상세

## 자격증
- Oracle Certified Master (OCM)

## 기술 스킬

### RDBMS
| 구분 | 상세 | 경험 |
|------|------|------|
| Oracle | 9i, 10g, 11g, 12c, 19c | 14년+ (설치/패치/이관/튜닝/운영) |
| PostgreSQL | 운영/관리 | 3년 (하나금융티아이 사내시스템) |
| MariaDB | 운영/관리 | 하나금융티아이 |
| Tibero | Oracle→Tibero 전환 | 삼성반도체 HR 시스템 |
| Altibase/Timestem | 인메모리 DB 운영 | KT |

### 고가용성 / 복제
- Oracle RAC 2~3노드 구축 및 운영 (Exadata 포함)
- Oracle Data Guard (Active Data Guard, DR 구축)
- Oracle GoldenGate (OGG) 실시간 복제 구성
- Exadata 운영 관리 (NS홈쇼핑)

### 백업 / 복구
- RMAN 백업 스케줄링 및 자동화
- RMAN Clone (테스트 DB 복제)
- 테이블 단위 Point-in-Time Recovery
- DataPump / TTS 기반 대용량 데이터 이관

### SQL 튜닝 / 성능 관리
- AWR / ASH / SQL Trace 기반 성능 분석
- 실행계획(Execution Plan) 분석 및 최적화
- 인덱스 설계 / 파티션 테이블 설계
- 성능 분석 리포트 작성 및 개선안 도출

### 관리 / 모니터링
- OEM Cloud Control 12c/13c 구축 및 운영
- Prometheus + Grafana 모니터링 대시보드 구축
- 메타 데이터 기반 표준화 관리
- 오브젝트 반영 / 권한 관리 / 감사 대응

### 클라우드 / 마이그레이션
- AWS Cloud DB 전환 (대한항공)
- U2L 마이그레이션 (Unix → Linux)
- Oracle → Tibero DB 전환 (삼성반도체)
- Oracle → PostgreSQL 마이그레이션 (ora2pg)
- 해외 법인 신규 DB 구축 (LG Display 중국)

### OS / 인프라
- Linux: RHEL, Rocky Linux, Oracle Linux
- VMware 가상화 환경 운영
- HAProxy 로드밸런서 구성
- SSH 보안 강화 / fail2ban 운영

### 스크립팅 / 자동화
- Python: 모니터링 자동화, 데이터 처리, 리포트 생성
- Bash Shell: cron 기반 운영 자동화 스크립트

### 산업 도메인 경험
- 금융: 하나금융티아이, 한화생명, ING생명
- 항공: 대한항공 (AWS 전환, DR 구축)
- 공공: 복지부, 우정국, 관세청, 국세청
- 제조: LG Display, 삼성반도체
- 유통: NS홈쇼핑 (Exadata), 롯데닷컴, 롯데면세점
- 통신: KT
- 미디어: 동아일보, 동아오츠카
