## 기술 노트 📝

### HQT (Heuristic Query Transformation) 핵심
- **비용 계산 없이** RULE 기반으로 SQL 변환
- **FK 제약조건이 JE(Join Elimination)에 필수**
- **NOT NULL 제약조건이 CNT 변환에 영향**
