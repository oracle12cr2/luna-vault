# Claude Cowork 완벽 활용 가이드: 일반인을 위한 일상 업무 자동화

> 출처: https://www.youtube.com/watch?v=mcJXTYb0Vt4

## 핵심
Claude Cowork = **비개발자용 Claude Code**. GUI로 파일 관리, 브라우저 제어, 외부 앱 연동을 자연어로 실행.

## 주요 기능 5가지

### 1. 파일 관리 & 다중 파일 생성
- 이미지 187개 자동 분류, 영수증 → 엑셀 정리
- 웹 리서치 → PDF/DOCX/PPTX 보고서 동시 생성
- "병렬로 수행해라"로 여러 작업 동시 실행 가능

### 2. 커넥터 (외부 앱 연동)
- Gmail, Google Drive, Google Calendar, Slack, Notion, Figma, Canva 등
- 예: Slack 대화록 분석 → PDF 저장 → 같은 채널로 재전송 → Notion에도 저장

### 3. 예정됨 (스케줄) — 신규 유료 기능
- 반복 작업을 정해진 시간에 자동 실행
- "매일 아침 9시 AI 뉴스레터 → Slack DM" 자동 배포
- 모델 선택 가능 (쉬운 건 Sonnet/Haiku, 어려운 건 Opus)
- 실행 빈도: 시간별/매일/주간 등 설정

### 4. 스킬 (반복 작업 모듈화)
- .md 파일로 워크플로우 저장
- 예: URL 하나 → 뉴스레터 + LinkedIn + Threads 한국어 동시 생성
- 생성 시 클로드가 출력 언어, 파일 형식, 스타일 등 질문 → 명확한 결과물 보장

### 5. 플러그인 (커넥터 + 스킬 번들)
- `/repurpose` → 콘텐츠 생성
- `/distribute` → 채널별 일괄 배포
- Anthropic 공식 플러그인 탐색 기능 제공
- 분야별 템플릿 확충 예정 (CS, 데이터분석, 금융, 법무, 마케팅)

## Claude Code vs Cowork
- **Claude Code**: 터미널 환경, 고도화된 소프트웨어 개발
- **Cowork**: GUI 환경, 직장인/일반인 반복 업무 자동화
