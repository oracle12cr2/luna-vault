## PostgreSQL 16 업그레이드 세부사항
- postgres01에 PostgreSQL 모듈 16 활성화: `dnf module enable postgresql:16`
- 패키지명: postgresql-server, postgresql-contrib (16 접미사 아님, 모듈로 버전 결정)
- 13→16 업그레이드: dnf가 자동으로 향상(upgrade) 처리
- 기존 데이터 디렉토리(/var/lib/pgsql/data)는 13 포맷 → initdb 재실행 필요할 수 있음
- blog_user 테이블 권한: GRANT ALL만으로 부족 → ALTER TABLE ... OWNER TO blog_user 필요했음
