# 미국 주식 무한 매수 자동매매 봇 (Claude Code + 7 프롬프트)

> 출처: https://www.youtube.com/watch?v=y6bUeOt5FB4

## 핵심 전략
극단적 과매도 구간(RSI 30 이하)에서 분할 매수하는 무한 매수 전략

## 사용 지표

### 기술적 지표
- **RSI**: 30 이하 = 과매도 → 매수 점수 높게 부여
- **MACD**: 골든크로스(단기>장기) = +15점, 데드크로스 = -10점
- **이동평균선**: 20일/50일 정배열(현재가 위) = 매수 우호

### 펀더멘탈 지표
- **PER**: 낮을수록 저평가
- **ROE**: 높을수록 수익성 좋음
- **매출성장률**: 높을수록 성장성

## 구현 흐름 (7 프롬프트)
1. 봇 뼈대 생성
2. yfinance 데이터 수집
3. 점수 산정 로직 (기준: 60점 이상 매수)
4. 페이퍼 트레이딩
5. 리포트 생성
6. 메인 실행
7. 테스트
- 보너스: Telegram 알림 연동

## 우리 시스템에 반영한 것
1. **MACD** → multifactor.py에 score_macd() 추가 (가중치 10%)
2. **펀더멘탈** → score_fundamental() 추가 (PER/ROE/매출성장률, 가중치 15%)
3. TB_FINANCIAL_STMT(380만건)에서 실제 재무 데이터 조회하여 반영
