# Claude AI 프로 트레이더 33가지 스킬 팩

> 출처: https://www.youtube.com/watch?v=6_9mJEq9na0
> 상태: GitHub 자료 요청 중 (받으면 스킬 분석 예정)

## 핵심 철학
- 감이 아닌 **정량 스코어링** (0~100점)
- **스킬 체이닝** — 여러 스킬을 워크플로우로 조합
- **자기 개선** — 품질 90점 미만이면 자동 코드 수정

## 인코딩된 투자 전략
- **윌리엄 오닐** — CAN SLIM (성장주 발굴)
- **마크 미너비니** — 슈퍼 퍼포먼스 (추세 추종)
- **스탠리 드러켄밀러** — 매크로 + 집중 투자

## 주요 스킬 (33개 중)
- `/us-stock-analyze` — 종목 분석 (시총, PE, EPS, 이평선, RSI, FCF)
- `/market-news-analyze` — 10일간 뉴스 → 핵심 이벤트 + 종목별 반응
- `/sector-rotation` — 섹터 로테이션
- `/earnings-analyzer` — 어닝 서프라이즈

## 비용 구조
- FMP API 무료: 하루 250회 → 18개 스킬 사용 가능
- 나머지 15개: 유료 API 필요
- 토큰 절감: Opus 대신 Sonnet 권장

## 우리 시스템 적용 가능 항목
1. **뉴스 → 종목 영향도 분석** (news_monitor.py 활용)
2. **어닝 서프라이즈 감지** (TB_FINANCIAL_STMT 380만건)
3. **자기 개선** (주간 리뷰 → 파라미터 자동 조정)
4. **스킬 체이닝** (이미 구현: 지표→스코어→판단→실행)
